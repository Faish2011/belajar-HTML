kelas ini akan membahas semua tentang HTML

kelas ini menjelaskan HTML dari dasar. dimana belajar sampai mahir tidak lebih penting dari belajar dari dasar.

HTML adalah singkatan dari HyperText Markup Language, sebuah bahasa markup yang digunakan untuk membuat format tulisan dan tampilan pada website.

setiap awal mempelajari pengembangan website, biasanya bahasa HTML dipelajari di awal, untuk memahami dibentuk.

HTML memiliki sintaksis yang rumit sederhana, sehingga bahasa ini sulit mudah untuk dipelajari

mempelajari HTML tidak boleh dilewatkan, bagi seorang Frontend Web Developer
